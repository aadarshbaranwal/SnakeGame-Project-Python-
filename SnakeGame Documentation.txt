Snake Game ‚Äì Documentation
The Snake Game is a classic arcade-style game developed using Python and Tkinter.
The player controls a snake that moves around the screen, eats food to grow longer, and earns points.
The game ends when the snake hits the wall or collides with itself.
This version includes:
Real-time score tracking
Keyboard-based controls
Restart and exit options
Dynamic snake color selection

Game Functions Explained
start_game()
Initializes the game state
Resets score
Clears canvas
Creates new snake and food
Starts the game loop
next_turn(snake, food)
Updates snake movement
Checks food collision
Grows snake or removes tail
Updates score
Schedules next movement using after()
change_direction(new_direction)
Updates snake direction
Prevents reversing into itself
check_collisions(snake)
Detects:
Wall collision
Self-collision
Returns True if collision occurs
game_over()
Stops the game loop
Displays GAME OVER text
Shows Restart and Exit buttons
on_exit()
Closes the game window safely
